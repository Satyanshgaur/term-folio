# Satyansh Gaur | AI Infrastructure Portfolio

A high-fidelity, workstation-style portfolio built with React, TypeScript, and Tailwind CSS. Featuring a premium glassmorphic aesthetic inspired by Tokyo Night and a fully interactive terminal environment.

## 🚀 Key Features

- **Full-Page Shell Environment**: Immersive terminal experience with persistent history and CRT glow effects.
- **AI Systems Branding**: Focus on hardware-aware software, CUDA kernels, and LLM inference.
- **Dynamic Workstation Layout**: Permanent system monitor sidebar with live GPU/VRAM metrics.
- **Dual Interface**: Seamlessly switch between `workspace.sh` (Shell) and `dashboard.gui` (Graphical View).
- **Rich Project Artifacts**: In-depth technical write-ups with architecture diagrams and performance benchmarks.

## 🛠 Tech Stack

- **Frontend**: React (Vite), TypeScript
- **Styling**: Tailwind CSS (Glassmorphism + Tokyo Night Palette)
- **CI/CD**: GitHub Actions
- **Deployment**: Optimized for Vercel
